# Simple Interest Calculator

## Description
This project is a simple interest calculator developed using Bash scripting. It calculates the simple interest based on the principal amount, annual rate of interest, and time period provided by the user.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Features
- Accepts user input for:
  - Principal amount
  - Rate of interest
  - Time period
- Calculates simple interest instantly.
- Simple and easy-to-understand Bash script.

## Files
- `simple-interest.sh` – Bash script to calculate simple interest.

## Requirements
- Linux, macOS, or Windows with Git Bash
- Bash Shell

## Usage

Clone the repository:

```bash
git clone https://github.com/Gnanasankar7/simple-interest-calculator.git
```

Go to the project directory:

```bash
cd simple-interest-calculator
```

Give execute permission:

```bash
chmod +x simple-interest.sh
```

Run the script:

```bash
./simple-interest.sh
```

## Example

```
Enter Principal:
10000
Enter Rate of Interest:
5
Enter Time (years):
2

Simple Interest = 1000
```
